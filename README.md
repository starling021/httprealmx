# httprealm
HTTP proxy for Realm of the Mad God written in Python 3.6

## Installation

To build HTTPRealm from sources, simply use this commands:

``` {.sourceCode .bash}
git clone git://github.com/crinny/httprealm
cd httprealm
pip install -r requirements.txt
```

That's it!
